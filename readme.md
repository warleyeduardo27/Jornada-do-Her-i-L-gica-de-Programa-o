🛡️ Jornada do Herói: Lógica de Programação
Este repositório contém os desafios de código desenvolvidos durante a formação de lógica de programação. O projeto simula um sistema de RPG onde é possível criar heróis de diferentes classes e gerenciar itens mágicos com atributos específicos.

🚀 Tecnologias Utilizadas
Linguagem: JavaScript (Node.js)

Ambiente de Testes: DIO Editor / W3Schools TryIt

Sistema Operacional: Ubuntu 24.04 LTS

🕹️ Funcionalidades
1. Sistema de Classes de Heróis
Uma classe genérica que representa um herói com propriedades de nome, idade e tipo.

Tipos suportados: Mago, Guerreiro, Monge, Ninja e Sereia.
